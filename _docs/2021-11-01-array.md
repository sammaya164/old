---

Array関数を使って配列を作成できる。

{% highlight vb %}
Dim fruit

fruit = Array("みかん", "りんご", "メロン")

MsgBox fruit(0) 'みかんと表示される
MsgBox fruit(1) 'りんごと表示される
MsgBox fruit(2) 'メロンと表示される
{% endhighlight %}

`Array()`とすれば要素をもたない配列も作成できる。

Array関数の引数と戻り値：

|引数|説明|
|---|---|
|引数|要素|
|戻り値|配列|
